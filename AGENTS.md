# Expo HAS CHANGED

Read the exact versioned docs at https://docs.expo.dev/versions/v54.0.0/ before writing any code.

This project is pinned to Expo SDK 54, not the latest SDK, because the App Store's public
Expo Go build has not yet been approved for newer SDKs (Apple review lag). Do not bump the
`expo` package version without checking that Expo Go on the App Store actually supports it.
